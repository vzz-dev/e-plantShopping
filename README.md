🌿 Paradise Nursery - Aplicación de compra de plantas

Este proyecto es una aplicacion de compras desarrollada con React para una tienda en linea de plantas llamada Paradise Nursey.

los usuarios pueden:

- Explorar diferentes categorias de plantas de interior
- Agregar plantas al carrito de compras
- Aumentar o disminuir la cantidad de productos
- Ver el costo total antes de realizar el pago

Este proyecto demuestra el uso de React y Redux para la gestión del estado y el desarrollo de interfaces de usuario.
